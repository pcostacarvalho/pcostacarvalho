# Hello, I'm Pamela 👋

I'm a computational physicist based in Uppsala, Sweden, with a growing focus on data science and machine learning applied to physical systems.

I hold a PhD in Physics and I'm currently a postdoctoral researcher at Uppsala University, where I develop ML models to predict and correct systematic errors in physics-based simulations of materials. I'm also finishing an MBA in Data Science and Analytics (ESALQ/USP), where my thesis applies GLM and tree-based regressors to a dataset I built and published myself.

My background spans first-principles simulations, high-throughput workflows, and predictive modelling — with a strong interest in bringing these skills to data-intensive problems in industry and research.

## Featured project

### 🔧 [EMTOFlow](https://github.com/pcostacarvalho/emtoflow)
A Python toolkit for automating high-throughput physics simulations on HPC clusters. Handles input generation, parameter optimization, SLURM job scheduling, and results extraction — all driven by a single YAML config file.
